## Description:

Intepreter for mathematical equations.
Based on https://ruslanspivak.com/lsbasi-part1/

## Supports:

### Operations

- Addition: +
- Subtraction: -
- Multiplication: *
- Division: /
- Modulo: %
- Parentheses: ( )

### Error handling

- Unexpected character
- Unexpected token
- Reached end of line while parsing
